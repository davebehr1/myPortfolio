DEVOPS

I have a keen interest in DevOps(developer operations) which takes into account software creation and deployment. Good software has no worth if it is not available and reliable, this is what the field of devops aims to provide, by firstly setting up a work flow that makes it easy for new features to be deployed to software and by keeping the service up and running constantly scaling it when necessary. My experience has mainly centered around a few technologies including Rancher and Docker which I use on a day to day basis, Github as a version control system and travis which is used for continous integration. I am constantly growing in this relatively new field and will do my best to provide the best service possible.
